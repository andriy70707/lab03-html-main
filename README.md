# Практична робота №3 — HTML + CSS + JavaScript сайт

##  Опис
Це навчальний багатосторінковий сайт, створений на основі HTML, CSS та JavaScript.
Проєкт демонструє базову структуру сайту, стилізацію інтерфейсу та інтерактивність.



##  Що реалізовано

###  HTML
- 3 сторінки: index.html, pages/about.html, pages/contact.html
- Семантична структура
- Навігація між сторінками
- Контент: заголовки, абзаци, списки, таблиця
- Медіа: зображення, відео, аудіо
- Форма з input, textarea, select, radio, checkbox



###  CSS
- Зовнішні файли стилів (style.css, responsive.css)
- Flexbox
- Адаптивність (media queries)
- Оформлення:
  - header / footer
  - меню
  - картки
  - таблиця
  - форма
- Hover ефекти та transition



### JavaScript
- Підключення через external script (main.js)
- Робота з DOM
- Події: click, input, submit, scroll

####  Реалізовані функції:
- Активне меню навігації
- Перемикач теми (збереження в localStorage)
- Кнопка "вгору"
- Динамічний рік у footer
- Акордеон (FAQ / навички)
- Модальне вікно
- Обробка кнопки (click)

####  Форма:
- Валідація (ім’я, email, повідомлення)
- Лічильник символів
- Збереження чернетки (localStorage)
- Відновлення даних після перезавантаження
- Обробка submit через FormData
- Вивід результату на сторінку



##  Як запустити
1. Відкрити папку проєкту у VS Code
2. Відкрити index.html через Live Server  
   або просто відкрити файл у браузері
